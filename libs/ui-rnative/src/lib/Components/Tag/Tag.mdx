## Setup

Install and set up the library with our [Setup Guide →](?path=/docs/getting-started-setup--docs).

### Basic Usage

```tsx
import { Tag } from '@ledgerhq/lumen-ui-rnative';

function MyComponent() {
  return <Tag size="md" appearance="accent" label="My Tag" />;
}
```

### Custom Styling

While the component comes with predefined styles, you can extend them using the `lx` prop for layout adjustments:

```tsx
<Tag
  size="md"
  appearance="accent"
  label="Custom Tag"
  lx={{ marginTop: 's4' }}
/>
```

### With Icons

The Tag component supports icons with automatic size adjustment based on the tag size:

```tsx
import { Tag } from '@ledgerhq/lumen-ui-rnative';
import { Check } from '@ledgerhq/lumen-ui-rnative/symbols';

function MyComponent() {
  return <Tag size="md" appearance="success" icon={Check} label="Success" />;
}
```
