# MediaCard

## Introduction

MediaCard is a promotional card component that displays a full-bleed background image with gradient overlays for text readability. It accepts text content, optional leading content (e.g. tags, icons), and a close button.

## Anatomy

<Canvas of={MediaCardStories.Base} />

- **MediaCard**: Root container with background image, gradient overlays, text, optional leading content, and close button
- **`text`**: Primary text content (clamps at 3 lines), supports rich text via inline elements
- **`leadingContent`**: Optional slot for secondary content above the text (tags, badges, icons)
- **Close button**: Rendered via the `onClose` prop (positioned absolute top-right)

## Accessibility

- The card root uses `role="button"` with keyboard activation (Enter/Space) and a visible focus ring
- The close button uses `aria-labelledby` to compose its accessible name from the card text (e.g. "Close, Black Friday sale")
- The background image and gradient overlays are marked `aria-hidden` since they are decorative
- The component forwards refs and spreads HTML attributes for ARIA support

## Basic Usage

```tsx
import { MediaCard, Tag } from '@ledgerhq/lumen-ui-react';

function MyComponent() {
  return (
    <MediaCard
      imageUrl='/promo.jpg'
      text='Card text'
      leadingContent={<Tag label='New' size='md' />}
      onClick={() => console.log('clicked')}
      onClose={() => console.log('closed')}
    />
  );
}
```

### Rich Text Content

Use inline elements in the `text` prop for mixed styling:

```tsx
<MediaCard
  imageUrl='/promo.jpg'
  text={
    <>
      <span>Black Friday sale. </span>
      <span className='text-interactive'>3 days with no fees </span>
      <span>on your transactions.</span>
    </>
  }
  onClick={() => {}}
  onClose={() => {}}
/>
```
