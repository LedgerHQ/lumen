# XAxis

The x-axis renders ticks, labels, an optional baseline and grid along the
horizontal edge of the chart's drawing area.

## Usage

`XAxis` is not used as a standalone component. Enable it with `showXAxis` and
configure it through the `xAxis` prop on `LineChart`.

`XAxisProps` extends the shared `BaseAxisProps` and adds `position` (`'top'` or
`'bottom'`).


## Grid

`showGrid` draws a vertical grid line at each tick. `gridLineStyle` toggles
between dashed and solid.
