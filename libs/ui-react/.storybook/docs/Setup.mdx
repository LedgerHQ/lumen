## 🛠️ Setup

### 📦 Available Packages

Our design system is available as npm packages:

- **[@ledgerhq/lumen-design-core](https://www.npmjs.com/package/@ledgerhq/lumen-design-core)** - Design tokens, themes, and core utilities all in one tailwind preset.
- **[@ledgerhq/lumen-ui-react](https://www.npmjs.com/package/@ledgerhq/lumen-ui-react)** - React components built with Tailwind CSS v4 and Radix UI primitives.
- **[@ledgerhq/lumen-ui-rnative](https://www.npmjs.com/package/@ledgerhq/lumen-ui-rnative)** - React Native components built with NativeWind.

### Steps

1 . **Load the Inter font family:**

Lumen typography uses the **Inter** font. You must load it in your project for text to render correctly.

**Via Google Fonts** (recommended) — add to your HTML `<head>`:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap"
  rel="stylesheet"
/>
```

**Or self-hosted** — download from [Google Fonts](https://fonts.google.com/specimen/Inter) and declare a `@font-face` in your CSS:

```css
@font-face {
  font-family: 'Inter';
  font-style: normal;
  font-display: swap;
  src: url('/fonts/inter.woff2') format('woff2');
}
```

2 . **Configure Tailwind:**
See our [Setup Tailwind](?path=/docs/getting-started-setup-tailwind--docs) guide for complete setup instructions.

3 . Install `lumen-ui-react` package:

Install the package:

```bash
npm install @ledgerhq/lumen-ui-react
```

And its peer dependencies:

```bash
npm install clsx tailwind-merge class-variance-authority
```

4 . Start using components:

```bash
# Install peer dependency related to the button component
npm install @radix-ui/react-slot
```

`@radix-ui/react-slot`: This dependency is used internally by the Button component to enable flexible composition patterns. It allows the Button to merge its props with child elements when needed.

```tsx
import { Button } from '@ledgerhq/lumen-ui-react';

function App() {
  return <Button appearance='base'>Get Started</Button>;
}
```

**5. Setup AI Assistant**

For better AI assistance (Cursor, Claude, VS Code Copilot, etc.), reference the included rules.

**For Cursor**, add to your `.cursorrules`:

```
@node_modules/@ledgerhq/lumen-ui-react/ai-rules/RULES.md
```

[Learn more about AI rules →](https://www.npmjs.com/package/@ledgerhq/lumen-ui-react#-ai-assistant-setup)

**Check out our [User Guide](https://github.com/LedgerHQ/lumen/blob/main/USER_GUIDE.md) for more information on how to use the UI Kit.**
