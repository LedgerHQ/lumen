# AvatarButton

<CustomTabs>
  <Tab label="Overview">

## Introduction

`AvatarButton` wraps an [`Avatar`](/docs/react-avatar--docs) in a semantic `<button>`, for use as an interactive trigger such as opening a user menu or navigating to a profile. It accepts every `Avatar` prop plus `onClick`.

## Fallback

Like `Avatar`, it falls back to initials or a User icon when no image is available.

<Canvas of={AvatarButtonStories.Fallback} />

## Accessibility

- Renders a real `<button type="button">`, so it is keyboard focusable and activatable out of the box.
- Provide a meaningful `alt` describing the action (e.g. `"Open user menu"`); it is exposed as the avatar's accessible label.
