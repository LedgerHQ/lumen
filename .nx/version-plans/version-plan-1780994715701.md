- fix(switch): fix reanmimated issue on the Switch + simplify implementation (removing Slot)

**Note about the reanimated issue on LW:**
The root cause: on iOS with the New Architecture (Fabric), Reanimated 4 doesn't apply an animated transform to an empty leaf Animated.View. The Switch thumb was exactly that — a childless Animated.View animating only translateX — so its transform was never applied (the thumb stayed put), while the track color still changed via normal React re-render. Android wasn't affected because Fabric's view-flattening/handling differs per platform.

Every other Lumen animation worked because their animated views had real children (PageIndicator strip, AmountInput text, Spin) or were special (the absolute TabBar pill).

The fix: give the thumb's Animated.View a real child (the drawn circle) so it's no longer an empty leaf. (Neither the global Reanimated feature flag nor collapsable={false} addressed it.)
