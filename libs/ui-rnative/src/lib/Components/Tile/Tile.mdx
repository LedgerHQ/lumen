## Introduction

Tile components are compact, card-like interactive elements designed for horizontal lists, carousels, or grids. They use a flexible composite pattern (same as Card) that allows you to compose tiles with leading content (e.g. Spot), text, and custom elements in a vertical layout.

> View in [Figma](https://www.figma.com/design/JxaLVMTWirCpU0rsbZ30k7/2.-Components-Library?node-id=5783-1328).

## Anatomy

- **TileHeader**: Container that lays out TileLeading and TileTrailing vertically (TileTrailing underneath TileLeading)
- **TileLeading**: Wraps leading content (e.g. Spot) + TileContent + any additional content (e.g. Tags), stacked vertically
- **TileContent**: Container for title and description
  - **TileTitle**: The main label of the item (required)
  - **TileDescription (optional)**: Provides additional context
- **TileTrailing (optional)**: Trailing content below the leading block (e.g. Tag, Button)

### Appearance & States

Two appearance variants are available with multiple interactive states:

**Appearances:**
- **no-background** (default): Transparent background with pressed states
- **card**: Surface background color with pressed states

**States:**
- **Default**: Base state with no interaction
- **Pressed**: Darker background when pressed
- **Disabled**: Non-interactive state with disabled styles

## Basic Usage

```tsx
import {
  Tile,
  TileHeader,
  TileLeading,
  TileContent,
  TileTitle,
  TileDescription,
  Spot,
} from '@ledgerhq/lumen-ui-rnative';
import { Settings } from '@ledgerhq/lumen-ui-rnative/symbols';

function MyComponent() {
  return (
    <Tile
      lx={{ width: 's128' }}
      onPress={() => console.log('Primary action')}
      onLongPress={() => console.log('Secondary action')}
    >
      <TileHeader>
        <TileLeading>
          <Spot appearance='icon' icon={Settings} />
          <TileContent>
            <TileTitle>Settings</TileTitle>
            <TileDescription>Manage preferences</TileDescription>
          </TileContent>
        </TileLeading>
      </TileHeader>
    </Tile>
  );
}
```

## Advanced Patterns

### With Trailing Content (Tags, etc.)

Place Tags or other content after TileContent inside TileLeading.

```tsx
import {
  Tile,
  TileHeader,
  TileLeading,
  TileContent,
  TileTitle,
  TileDescription,
  Tag,
  Spot,
} from '@ledgerhq/lumen-ui-rnative';
import { Bitcoin } from '@ledgerhq/lumen-ui-rnative/symbols';

<Tile appearance='card'>
  <TileHeader>
    <TileLeading>
      <Spot appearance='icon' icon={Bitcoin} />
      <TileContent>
        <TileTitle>Bitcoin</TileTitle>
        <TileDescription>BTC</TileDescription>
      </TileContent>
      <Tag label='Active' appearance='success' />
    </TileLeading>
  </TileHeader>
</Tile>;
```


### Centered Content

Use the `centered` prop to vertically center the tile content. This is useful when a tile has less content than its neighbors in a grid, like a "Show more" tile that needs to align with richer tiles.

```tsx
import { ChevronRight, CryptoBitcoinCoin } from '@ledgerhq/lumen-ui-rnative/symbols';
import { Spot } from '@ledgerhq/lumen-ui-rnative';
import { View } from 'react-native';

// In a grid where other tiles have more content
<View style={{ flexDirection: 'row', gap: 8 }}>
  <Tile appearance='card'>
    <TileHeader>
      <TileLeading>
        <Spot appearance='icon' icon={CryptoBitcoinCoin} />
        <TileContent>
          <TileTitle>Title</TileTitle>
          <TileDescription>Description</TileDescription>
        </TileContent>
        <Tag label='Label' />
      </TileLeading>
    </TileHeader>
  </Tile>

  {/* Show more tile with centered content */}
  <Tile appearance='card' centered>
    <TileHeader>
      <TileLeading>
        <Spot appearance='icon' icon={ChevronRight} />
        <TileContent>
          <TileTitle>Show more</TileTitle>
        </TileContent>
      </TileLeading>
    </TileHeader>
  </Tile>
</View>
```

### Horizontal List

```tsx
import { ScrollView } from 'react-native';
import { Tile, TileHeader, TileLeading, TileContent, TileTitle, TileDescription, Spot } from '@ledgerhq/lumen-ui-rnative';

<ScrollView horizontal>
  {items.map((item) => {
    const Icon = item.icon;
    return (
      <Tile key={item.id} lx={{ width: 's128', flexShrink: 0 }}>
        <TileHeader>
          <TileLeading>
            <Spot appearance='icon' icon={Icon} />
            <TileContent>
              <TileTitle>{item.title}</TileTitle>
              <TileDescription>{item.description}</TileDescription>
            </TileContent>
          </TileLeading>
        </TileHeader>
      </Tile>
    );
  })}
</ScrollView>;
```
