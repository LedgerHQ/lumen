# DataTable

A compound component that combines TanStack Table with the Lumen `Table` primitives.
Wrap a `<DataTable />` inside a `<DataTableRoot>` to render a fully featured data table from a TanStack table instance.

## With Infinite Loading

Combine properties `onScrollBottom` with TanStack Query's `useInfiniteQuery` to load data as the user scrolls to the bottom of the table.
Set `pagination='infinite-scroll'` and `loading=true` on `DataTableRoot` to enable infinite scroll indicator.
