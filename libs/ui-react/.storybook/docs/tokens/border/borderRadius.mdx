# Border Radius

Border radius defines the curvature of UI elements, ensuring consistency and visual harmony across our products.

## ⚡ Structure

All radius tokens follow the structure: `border-radius-size`

Example:

- `border-radius-md`
- `border-radius-lg`

<img src={borderRadiusTaxonomy} alt="Border radius token taxonomy structure" />

<Markdown>
  {`
| Element  | Definition | Example |
| -------- | ---------- | ------- |
| Property | border-radius token determining the curvature of elements | border-radius |
| Size     | defines the curvature of an element's corners | none, xxs, xs, sm, md, lg, xl, 2xl, full |
`}
