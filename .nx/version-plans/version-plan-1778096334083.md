Sync Figma Design Tokens & Symbols

## New tokens

Added the following semantic color tokens across all themes (enterprise, ledger-live, websites):

**Background**
- `background-canvas-overlay-chart`
- `background-grey`
- `background-muted-transparent-0`
- `background-error-transparent-0`
- `background-success-transparent-0`

**Border**
- `border-muted-transparent-0`
- `border-muted-subtle-transparent`
- `border-error-transparent`
- `border-success-transparent`

## Breaking change

The `bg-gradient-error`, `bg-gradient-success`, and `bg-gradient-muted` Tailwind utilities now reference the new `*-transparent-0` tokens (`--background-error-transparent-0`, `--background-success-transparent-0`, `--background-muted-transparent-0`) instead of the `*-transparent` variants for their fully-transparent gradient stop. Consumers relying on the previous CSS variable references must update to the new token names.
