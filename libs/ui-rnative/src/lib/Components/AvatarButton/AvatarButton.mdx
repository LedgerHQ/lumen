# AvatarButton

## Introduction

`AvatarButton` wraps an [`Avatar`](/docs/rnative-avatar--docs) in a `Pressable`, for use as an interactive trigger such as opening a user menu or navigating to a profile. It accepts every `Avatar` prop plus `onPress` and `onLongPress`.

## Fallback

Like `Avatar`, it falls back to a User icon when no image is available.

<Canvas of={AvatarButtonStories.Fallback} />

## Disabled

Pass `disabled` to prevent interaction. The pressable is not activatable, and the avatar renders at reduced opacity.

<Canvas of={AvatarButtonStories.Disabled} />

## Accessibility

- Renders with `accessibilityRole="button"` so it is announced as a button by screen readers.
- Provide a meaningful `alt` describing the action (e.g. `"Open user menu"`); it is exposed as the avatar's accessible label.

## Basic Usage

function MyComponent() {
  return (
    <AvatarButton
      src="https://example.com/photo.jpg"
      alt="Open user menu"
      onPress={openMenu}
    />
  );
}
