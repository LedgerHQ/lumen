# MediaTag

## Introduction

MediaTag is a compact label used to categorize, classify, or highlight information alongside a required media element such as an image or crypto icon. Like [Tag](/docs/communication-tag--docs), it supports multiple appearances and sizes, but is designed for cases where the icon is richer than a Symbol.

### Size

MediaTags come in two different sizes:

- **md** (default) — the `icon` media should be rendered at **16px**.
- **sm** — the `icon` media should be rendered at **12px**.

The MediaTag does not resize its `icon`; consumers are responsible for passing a media element sized to match the tag's `size`.

<Canvas of={MediaTagStories.SizeShowcase} />

### Truncation

When a MediaTag's label exceeds the available space, the text is automatically truncated with an ellipsis.

## Setup

Install and set up the library with our [Setup Guide →](?path=/docs/getting-started-setup--docs).

### Basic Usage

Match the media element size to the MediaTag `size`: **16px** for `md`, **12px** for `sm`.

```tsx
import { MediaTag, MediaImage } from '@ledgerhq/lumen-ui-rnative';

function MyComponent() {
  return (
    <>
      {/* md → image size 16 */}
      <MediaTag
        size="md"
        appearance="accent"
        label="Ethereum"
        icon={
          <MediaImage
            src="https://crypto-icons.ledger.com/ETH.png"
            alt="Ethereum"
            size={16}
            shape="square"
          />
        }
      />

      {/* sm → image size 12 */}
      <MediaTag
        size="sm"
        appearance="accent"
        label="Ethereum"
        icon={
          <MediaImage
            src="https://crypto-icons.ledger.com/ETH.png"
            alt="Ethereum"
            size={12}
            shape="square"
          />
        }
      />
    </>
  );
}
```

### Custom Styling

While the component comes with predefined styles, you can extend them using the `lx` prop for layout adjustments:

```tsx
<MediaTag
  size="md"
  appearance="accent"
  label="Custom MediaTag"
  icon={myIcon}
  lx={{ marginTop: 's4' }}
/>
```

### With Crypto Icons

The MediaTag accepts any `ReactNode` as its icon, including crypto icons. Use **16px** for `md` and **12px** for `sm`:

```tsx
import { MediaTag } from '@ledgerhq/lumen-ui-rnative';
import CryptoIcon from '@ledgerhq/crypto-icons/native';

function MyComponent() {
  return (
    <>
      {/* md → icon size 16 */}
      <MediaTag
        size="md"
        label="Bitcoin"
        icon={<CryptoIcon ledgerId="bitcoin" ticker="BTC" size={16} />}
      />

      {/* sm → icon size 12 */}
      <MediaTag
        size="sm"
        label="Bitcoin"
        icon={<CryptoIcon ledgerId="bitcoin" ticker="BTC" size={12} />}
      />
    </>
  );
}
```
