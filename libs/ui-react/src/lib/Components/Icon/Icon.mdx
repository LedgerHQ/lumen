## 📏 Sizing

Our icons are designed to be scalable and are available in 7 standard sizes to fit various contexts within the UI.

The available sizes are: **16px, 20px, 24px, 32px, 40px, 48px, and 56px**

The thickness of each icon is automatically managed by our token system when you apply a size variable, ensuring visual consistency at any scale.

## Basic Usage

Import and use specific icons from the symbols library:

```tsx
import { ArrowUp, Settings, Plus } from '@ledgerhq/lumen-ui-react/symbols';

function MyComponent() {
  return (
    <div>
      <ArrowUp size={24} />
      <Settings size={20} />
      <Plus size={16} />
    </div>
  );
}
```

### Size Variants

Icons support predefined size variants that ensure consistent visual hierarchy:

```tsx
import { ArrowUp } from '@ledgerhq/lumen-ui-react/symbols';

function IconSizes() {
  return (
    <div className='flex items-center gap-4'>
      <ArrowUp size={16} />
      <ArrowUp size={20} />
      <ArrowUp size={24} /> {/* Default size */}
      <ArrowUp size={40} />
      <ArrowUp size={48} />
      <ArrowUp size={56} />
    </div>
  );
}
```

### Within Buttons

The Button component supports icons with automatic size adjustment based on the button size:

```tsx
import { Button } from '@ledgerhq/lumen-ui-react';
import { ArrowRight } from '@ledgerhq/lumen-ui-react/symbols';

function MyComponent() {
  return (
    <Button size='m' appearance='base' icon={ArrowRight}>
      Continue
    </Button>
  );
}
```

### Accessibility

Always provide proper accessibility attributes for interactive icons:

```tsx
import { Information } from '@ledgerhq/lumen-ui-react/symbols';

function AccessibleIcon() {
  return <Information size={20} aria-label='Account verification required' />;
}
```

### With Semantic Colors

Icons inherit the current text color and can be styled with semantic color utilities:

```tsx
import {
  CheckmarkCircle,
  DeleteCircle,
  Clock,
} from '@ledgerhq/lumen-ui-react/symbols';

function SemanticIcons() {
  return (
    <div className='flex gap-4'>
      <CheckmarkCircle size={24} className='text-success' />
      <DeleteCircle size={24} className='text-error' />
      <Clock size={24} className='text-warning' />
    </div>
  );
}
```
