# Context7 MCP Usage Guidelines

Context7 provides up-to-date documentation for libraries and frameworks. Use it when you need accurate, current information about external dependencies.

**ALWAYS use Context7 when:**

- Researching how to use a library or framework (e.g., React, Radix UI, Shadcn, Tailwind CSS)
- Looking up API documentation for external packages
- Finding best practices for third-party libraries
- Verifying current syntax or feature availability
- You're unsure about implementation details of external dependencies

**Do NOT use Context7 for:**

- Internal codebase questions (use the search tools instead)
- Project-specific patterns or conventions
- Questions about our design system tokens
