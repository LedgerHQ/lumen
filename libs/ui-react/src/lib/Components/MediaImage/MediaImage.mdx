# MediaImage

<CustomTabs>
  <Tab label="Overview">

## Introduction

MediaImage displays an image with consistent sizing and shape. When the image fails to load or no source is provided, a background placeholder is shown automatically.

> View in [Figma](https://www.figma.com/design/JxaLVMTWirCpU0rsbZ30k7/2.-Components-Library).

## Anatomy

<Canvas of={MediaImageStories.Base} />

- **Container**: Sized wrapper with rounded corners and overflow clipping
- **Image**: Fills the container using `object-cover`
- **Fallback**: Background placeholder shown on missing or broken source

### Sizes

Eight sizes are available (12, 16, 20, 24, 32, 40, 48, 56). Border radius scales with size.

<Canvas of={MediaImageStories.SizeShowcase} />

### Shapes

<Canvas of={MediaImageStories.ShapeShowcase} />

- **square** (default): Rounded corners that scale with size
- **circle**: Fully rounded

## Accessibility

- The root element uses `role="img"` with `aria-label` derived from `alt`.
- The inner `<img>` is marked `aria-hidden` to avoid duplicate announcements.
- Always provide a meaningful `alt` prop so screen readers can announce the image.

## Basic Usage

```tsx
import { MediaImage } from '@ledgerhq/lumen-ui-react';

function MyComponent() {
  return <MediaImage src='https://example.com/icon.png' alt='Bitcoin' size={32} />;
}
```

### With Circle Shape

```tsx
<MediaImage src='https://example.com/icon.png' alt='Ethereum' size={48} shape='circle' />
```

### Lazy Loading

Defer image loading until the element is near the viewport:

```tsx
<MediaImage src='https://example.com/icon.png' alt='Token' imgLoading='lazy' />
```
