## Introduction

Tags are compact labels used to categorize, highlight status, or surface metadata. They provide quick visual context without drawing focus away from primary content.

> View in [Figma](https://www.figma.com/design/JxaLVMTWirCpU0rsbZ30k7/2.-Components-Library?node-id=722-5547&m=dev).

## Anatomy

- **Icon (optional):** An optional icon displayed before the label.

## Basic Usage

```tsx
import { Tag } from '@ledgerhq/lumen-ui-react';

function MyComponent() {
  return <Tag size='md' appearance='accent' label='My Tag' />;
}
```

### Custom Styling

While the component comes with predefined styles, you can extend them using Tailwind classes:

```tsx
<Tag
  size='md'
  appearance='accent'
  label='Custom Tag'
  className='mt-4' // Custom margin
/>
```

### With Icons

The Tag component supports icons with automatic size adjustment based on the tag size:

```tsx
import { Tag } from '@ledgerhq/lumen-ui-react';
import { Check } from '@ledgerhq/lumen-ui-react/symbols';

function MyComponent() {
  return <Tag size='md' appearance='success' icon={Check} label='Success' />;
}
```
