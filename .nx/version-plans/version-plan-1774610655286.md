BREAKING CHANGE(ListItem): remove `ListItemSpot`, `ListItemIcon`, and `ListItemTruncate`

### Migration

- `<ListItemSpot />` → `<Spot />` (disabled state still inherited via context)
- `<ListItemIcon icon={Wallet} />` → `<Wallet size={24} />`
- `<ListItemTruncate>` → `<ListItemContentRow>` for inline composition (e.g. title + Tag):

```diff
  <ListItemContent>
-   <ListItemDescription>
-     <ListItemTruncate>Text</ListItemTruncate>
-     <Tag label="New" appearance="accent" size="sm" />
-   </ListItemDescription>
+   <ListItemContentRow>
+     <ListItemDescription>Text</ListItemDescription>
+     <Tag label="New" appearance="accent" size="sm" />
+   </ListItemContentRow>
  </ListItemContent>
```
