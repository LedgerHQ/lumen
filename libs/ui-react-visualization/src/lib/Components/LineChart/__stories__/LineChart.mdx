# LineChart

A flexible line chart for displaying data trends over time. Supports multiple
series, area fills, axes, annotations and interactive scrubbing.

## Usage

`LineChart` is the entry point for the visualization library. It renders one
line per series across a single x/y axis pair, drawn as an SVG. It owns the data
and computes the scales; axes are configured through props (`xAxis` / `yAxis`),
while points, reference lines and the scrubber compose as children.

Under the hood it maps **data space** (your numbers) into **pixel space** (the
SVG) through a few coordinated pieces:

- **Series** ‚Äî your data, one line each.
- **Drawing area** ‚Äî the plotted region, derived from `width`/`height` minus
  `inset` and any axis padding.
- **Scales & domains** ‚Äî how data values map onto the drawing area, configured
  per axis.
- **Overlays** ‚Äî `Point`, `ReferenceLine` and `Scrubber` children, positioned in
  data space and drawn on top of the lines.

## Data

Each series' `data` array defines the y-values, one per index. `null` entries
render as gaps in the line.

<Canvas of={LineChartStories.MissingData} />

### Scales

Axes use a `linear` scale by default. Switch the algorithm with `scaleType`
(e.g. `'log'` for data spanning orders of magnitude, or `'band'` for discrete
categories). See the **XAxis** and **YAxis** pages for the full set of scale and
domain options.

## Sizing

`width` accepts a number (pixels) or a string like `'100%'` to fill the
container; `height` is always in pixels.

<Canvas of={LineChartStories.WidthAndHeight} />

`inset` reserves padding between the SVG edge and the drawing area ‚Äî the region
where data is actually plotted. A number applies uniformly; a partial object
overrides individual sides. Use it to leave room for overlays such as point
labels that would otherwise clip at the edges. In the examples below the border
outlines the full chart box, so the inset is the gap between it and the line.

<Canvas of={LineChartStories.Inset} />

## Interaction

Charts have built-in scrubbing, enabled with `enableScrubbing`. Listen for
changes through `onScrubberPositionChange`, and add a `<Scrubber>` child to
visualise the current position.

<Canvas of={LineChartStories.Scrubbing} />

See the **Scrubber** page for tooltips, beacons and magnetic points.

## Styling

### Area

`showArea` fills the region under each line; `areaType='gradient'` (default)
fades from the stroke color to transparent.
