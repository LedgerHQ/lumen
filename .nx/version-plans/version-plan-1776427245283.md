BREAKING_CHANGE(ui-react): remove SelectTriggerButton + use MediaButton directly

The `SelectMediaButton` preset component has been removed. Use `MediaButton` directly inside `SelectTrigger`'s `render` prop instead.

```diff
- import { SelectMediaButton } from '@ledgerhq/lumen-ui-react';
+ import { MediaButton } from '@ledgerhq/lumen-ui-react';

  <SelectTrigger
-   render={(renderProps) => (
-     <SelectMediaButton {...renderProps} label='All accounts' />
+   render={({ selectedValue, selectedContent }) => (
+     <MediaButton>
+       {selectedValue ? selectedContent : 'All accounts'}
+     </MediaButton>
    )}
  />
```
